---

This project tested on Macos and Windows.

### Setup
Install with npm
```
npm install -g android-dpi
```

### Usage 
All image file on folder
```
android-dpi .
```

Specified image
```
android-dpi foto-ktp.jpg
```

Specified multiple image
```
android-dpi meguri.jpg ai_uehara.jpg
```

### Supported format
```
['.jpg', '.jpeg', '.png', '.webp']
```
